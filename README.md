# SpendWise

A personal budget and expense tracking web application built for COMP250.

## Overview

SpendWise helps users manage their finances by tracking income and expenses, setting budgets, and visualising spending patterns over time. All data is stored locally in the browser using PouchDB — no server or account required.

## Features

- Add and categorise income and expense transactions
- Pre-filled budget page with common spending categories
- View transaction history and running balance
- Local data persistence with PouchDB
- Clean, responsive UI

## Tech Stack

- HTML, CSS, JavaScript
- PouchDB (local browser storage)

## Getting Started

2. Open `index.html` in your browser
